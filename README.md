# 医院管理系统简介 java1088

## 项目概述

医院管理系统是一款基于前后端分离架构的信息管理系统，旨在为医疗机构提供全面、高效、便捷的管理解决方案。

## 技术栈

### 前端技术

- Vue
- Element-UI

### 后端技术

- Spring Boot
- MyBatis Plus

### 数据库

- MySQL

## 功能模块

系统包含但不限于以下功能模块：

- 病患管理
- 医生排班
- 药品库存
- 费用结算
- 电子病历

## 系统角色

系统预设以下角色：

- 系统管理员
- 医生
- 药剂师
- 挂号收费员

## 运行环境

- 前端：现代浏览器
- 后端：Java 1.8及以上版本

## 部署步骤

1. 环境准备：确保已安装Java和MySQL。
2. 数据库部署：运行SQL脚本以创建数据库和表。
3. 后端部署：构建后端项目，部署到Java应用服务器。
4. 前端部署：构建前端项目，部署到Web服务器。

## 目录结构

```
医院管理系统
├── 前端代码
└── 后端代码
    ├── src
    ├── config
    └── sql
```
